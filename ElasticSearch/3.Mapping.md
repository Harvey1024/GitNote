# 3 Index
## 创建索引：
PUT /my_index {"settings": {}, "mappings": {}}  
创建索引后，默认新的字段会通过动态映射的方式添加到类型映射，可以通过配置：  
action.auto_create_index: false  
关闭自动创建，此时需要手动创建索引
## 删除索引
DELETE /my_index
## 索引设置
```json
put /my_index 
{
    "settings": {
        "number_of_shards": 1,
        "number_of_replicas": 0
    }
}
```
number_of_shards每个索引的主分片数，默认5
number_of_replicas每个索引的主分片副本数，默认1
